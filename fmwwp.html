# 1.视频压缩

步骤：

  1. 【选择文件】

    支持多选，多格式列队处理，比如同时选择视频、音频、图片文件

  2. 【保存目录】

    默认保存到桌面，注意：有部分功能会直接覆盖目录下同名文件，请谨慎！

  3. 【批量设置】

    此操作可跳过，使用默认参数也是推荐的。

    详细：

    • 压缩速度级别：音频和视频处理时，级别越快，需要的时间越短，但处理的效果越差。（实际上花费的时间的差别不明显，所以建议就保持默认即可）。
    • 宽度/高度上限：即允许的最大宽/高。大于此上限时，会等比缩放；小此上限则不做处理。
    • 大小上限：同理。这里对针的只是视频，其他文件类型无效。另外，处理的结果与此值是有一定误差的。
    • 输出名称：用于序列化文件名，让文件名有规律。按钮【复制】、【移动】的作用是不改变用于压缩的默认参数而只序列化文件，这在批量处理一些名称很乱的文件时很有用，比如要做成gif的大量序列图。
    • 按钮：【确定】->修改所有被锁定的文件。【重置】->恢复默认。【取消】->不进行任何操作。

  4. 【文件预览面板设置】

    播放、暂停按钮：当音频或视频文件支持播放预览时，此按钮会显示，点击可播放文件，再点击预览图即暂停。

    图标按钮：【锁定】->用于方便针对部分文件进行批处理设置。准确的说应该叫标记，即标记可以被批处理设置参数影响的文件。【显示/隐藏alpha通道】->显示/隐藏alpha通道，此按钮只是为了方便观察文件而已，几乎不用。【删除】->删除文件。注：以上三个按钮左击影响自身，右击则影响所有文件。

    时间控制滑杆：只对有时间长短的音频、视频有用。滑动它可以预览视频任意时间的图像。

    微调：有时候时间控制滑杆无法精准的取到需要的位置，微调可以以需要的帧数前进或后退。在精准截取视频片段时，非常有用。

    输入【序列图】项：用于将序列图合成gif动图或视频。勾选时，说明此文件是序列图的代表图（即第一张）。序列图的名称必须符合规律，否则无效。

    输出面板：

    • 重置：将当前文件中的参数重置到最初状态。
    • 文件名：输出名称，不要写后缀（扩展名）。
    • 格式：选择需要输出的文件类型的扩展名。
    • 大小：输出的大小，对结果质量直接的影响，只对视频有效。结果有误差。
    • 截取：点击后，会获取时间控制滑杆当前的时间位置。通过方法，可以设置截取的起点和终点。
    • 预览图：选择是否同时生成视频的预览图，用作网页上的视频封面。勾选后，即可选择生成预览图的时间位置，以获得最佳的封面图。
    • 分辨率：即视频或图片的长宽像素值。

  5. 【开始】

    即开始处理文件。处理视频时，程序比较消耗电脑CPU和内存，所以建议在处理过程中，不要打开太多程序，以使它处理更快。

    如果在处理之前未选择输出目录，默认会保存到电脑桌面，以下其他输出与此相同。

# 2.视频转音频

步骤：

同上。在【文件预览面板设置】中，选择输出需要的音频格式即可。

# 3.视频转gif动图

步骤：

同上。在【文件预览面板设置】中，选择输出格式为gif即可。

# 4.视频转单张图片

步骤：

同上。在【文件预览面板设置】中，将截取时间的起点和终点同时取到需要的时间位置，然后选择输出的格式为需要的图片格式即可。

# 5.序列图转视频

步骤：

同上。在【文件预览面板设置】中，勾选序列图，选择输出格式为需要的视频格式即可。

# 6.序列图转gif动图

步骤：

同上。在【文件预览面板设置】中，勾选序列图，选择输出格式为gif即可。

# 7.屏幕录制

步骤：

  1. 选择录制模式：
    • 全屏有声视频
    • 小屏有声视频
    • 全屏无声视频
    • 小屏无声视频
    • 无画面仅录音
  2. 视频输出帧速率：一般15至60。默认25即可。
  3. 视频输出比特率：视频的比特率，直接影响视频高清度和文件大小。不了解参数则保持默认即可。
  4. 音频输出比特率：音频的比特率，直接影响音频的质量和文件大小。不了解参数则保持默认即可。
  5. 音频录制设备：用于录制声音的设备，需要点击后面的按钮来检测电脑上是否有可用的录音设备。一般录麦克风的，或者录电脑内部的。如果没有，则可能电脑未安装音频驱动（如：Realtek High Definition），或未启用录音设备。
  6. 输出后匹配宽度：其实相当于规定视频输出的宽度上限。
  7. 录制区域：非全屏录制时，此按钮将被激活，可以点击它来设置需要录制的区域。有时候可能因为选择区域不符合视频标准，输出的视频在某些播放器可能不支持，此时可以再将此视频压缩一次试试。
  8. 点击【开始录制】：此时，本软件将全部隐藏，只有任务栏有选择卡。按快捷键【F2】将停止录制。

# 8.图片拼接

步骤：

  1. 排列方式：垂直布局方式与图片对齐方式。
  2. 高级模式：需要懂css样式表代码才能较好的操作。
  3. 预览：勾选时，即可看到布局。
  4. 确定即输出合成后的图片。
